# Node-Calendar-Backend

This project was generated with Express 4.18.2. It registers events on Mongo DB. It's used by the following GUI [Node Calendar Frontend](https://github.com/juanfix/react-calendar-frontend).

## Development server

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Rename the .env.template file to .env
4. Assign environment variables in the .env file.
5. Run `npm run dev` to init a dev server.
6. The appication will automatically reload if you change any of the source files.
